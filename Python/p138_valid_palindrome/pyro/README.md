## 문자열 슬라이싱

문자열을 자르는(slice) 행위이다.

파이썬에서는 말도 안되는 기능을 제공한다.

|안|녕|하|세|요|
|---:|---:|---:|---:|---:|
|0|1|2|3|4|
|-5|-4|-3|-2|-1|

```python
s = "안녕하세요"

s[:] == "안녕하세요" # 사본 리턴

s[1:4] == "녕하세"
s[1:-2] == "녕하"
s[1:] == "녕하세요"
s[1:100] == "녕하세요" # 인덱스가 크면, 문자열의 최대 길이만큼 표현

s[-1] == "요"
s[-4] == "녕"
s[:-3] == "안녕"
s[-3:] == "하세요"

s[::1] == "안녕하세요"
s[::-1] == "요세하녕안"
s[::2] == "안하요" # 2칸씩 이동
```
